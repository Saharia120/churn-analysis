Customer Churn Analysis

This repository contains an exploratory analysis of customer churn to understand the factors that influence whether customers stay with a service or leave. The goal of this analysis is to identify meaningful patterns within the dataset and highlight insights that can help improve customer retention strategies.

The notebook walks through data preparation, feature exploration, and visual analysis. Instead of focusing on predictive modeling, this project emphasizes understanding the relationships between customer demographics, service usage patterns, and churn behavior.
